# Gestion_incidents
# Incident Management System

Projet microservices réalisé dans le module Génie Logiciel

## Architecture

Services :

- Eureka Server (Service Discovery)
- Config Server (Configuration centralisée)
- API Gateway
- User Service
- Incident Service
- Comment Service
- Chat Service

## Technologies

- Spring Boot
- Spring Cloud
- React
- Docker
- Keycloak
- PostgreSQL

## Lancer le projet

docker compose up
